=======
whoyeah
=======
a grep like text finder in Python


Inspired
========
* grep-ack - a perl grep-like
* grin from http://pypi.python.org/pypi/grin for is_binary()

Feature
=======
* summery count result
* colorful
* ignore binary
* customize ignore list: .wyignore
